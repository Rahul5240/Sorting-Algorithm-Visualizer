# Sorting-Algorithm-Visualizer
This project is a simple Python-based Sorting Algorithm Visualizer that demonstrates how different sorting algorithms work step-by-step in the console. It helps users understand the internal workings and behavior of algorithms through real-time updates.

<br>
The program generates a random array and allows the user to choose a sorting algorithm. It then displays each step of the sorting process with a delay, making it easier to visualize how elements are compared and swapped.

##  Features
- Visualizes sorting step-by-step in the console
- Supports multiple algorithms:
  - Bubble Sort
  - Selection Sort
  - Merge Sort
- Adjustable speed for visualization
- Random array generation
- Easy to understand and beginner-friendly

# Technologies Used
- Python
- Standard Libraries (random, time)
